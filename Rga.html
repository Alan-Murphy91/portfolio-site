BJJ Interval Timer

This is a custom interval timer for Brazilian Jiu Jitsu that I designed for Royal Grappling academy, of which I am a member. During practice there are one-to-one sessions of sparring and intervals in which practitioners form groups and take turns defending or attacking different positions. The goal of this timer is to allow for quick and obvious understanding of which phase is currently live.

I used some simple jQuery for the timer application and used flexbox for the styling. Please note that this application was purpose built for a large wall-mounted display, as such it will not appear correctly on monitors less than 24" in width.
