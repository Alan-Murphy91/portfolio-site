Crime in Vancouver - Data Visualisation

This project is a visualisation of the crime rate in Vancouver in 2016. An interactive visualisation with a map allowing the user to focus on any metric they wish with associated geography. Thanks to kaggle I was able to obtain a CSV file with all the details. I imported this into a MongoDB library and used d3.js, dc.js and crossfilter.js to create the data visualisations.

Thanks to leaflet.js there is a data layer bound to the filtered data. This element is crucial as when the incidences of crime and their location are viewed on a map the data is understood more effectively, at least in my opinion.

This app is powered by Flask, a lightweight full stack python framework. It does not come out of the box with much unlike Django which make it ideal for simple applications such as this, however this project necessitated the use of two other applications for the databasing. Heroku, where it is hosted and MongoDB.

Please allow a few moments for the data to propagate on load!
