Space Invaders

Block Invaders! A duller take on the original premise because I was afraid of using sprites for some reason. This is a simple little object-oriented programming project that uses the p5js library, acting on the javascript canvas element. There is a setup loop which runs ones and then another loop which executes every milisecond

There were sound effects initially which ill re-include when I host the project here. If I get some time to extend the project it would be fun to include a boss level. Enemy rate of fire and speed of descension increases with each level.

Due to the nature of the project it is NOT RESPONSIVE.
