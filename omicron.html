Omicron - a Thematic Front End Project

This project a highly thematic front-end brochure website of a fictitious entity 'Omicron Technologies' that offer interstellar private charter. The website is built on CSS Grid and uses Alvaro Trigo's fullpage scroll plugin.

I experimented with different styles on the responsive view and used parallax backgrounds and different sized panels. CSS Grid made positioning the elements of the 'Our Most Popular Destinations' section very straightforward.
