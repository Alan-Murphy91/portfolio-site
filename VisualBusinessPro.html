Visual Business Pro

VBP is a django-based solution for turning company-centric database records into interactive data visualisations. An administrator can create an account and add a variety of records such as invoicing, credit control and employee information. The data is then rendered live on screen and can be filtered such as to allow the administrator to be very exacting in their data analysis.

Django comes out of the box with a SQLite database which eliminates any additional task of connecting an external database software to the application. The core of the application revolves around using Django's RESTful API backend framework to allow for the SQL data to be JSON-serialisable and available in a GET request. This data is driven through d3.js, dc.js and crossfilter.js to create the front-end interface. Bootstrap's grid system was used for the columns and custom CSS styling was used for the DC library aswell as the general site files to achieve the dark theme which I feel matches the colourful visualisations well. The to-do list for the app is an extension of the current features to replicate a modern accounts package and an attempt to further decouple the individual apps that compose the project.
